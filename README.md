### 项目简介

这是一个用来写学习笔记的博客系统。采用前后端分类架构，暂未考虑SEO。客户端使用Vue3开发、管理端使用React开发、后台使用Egg开发。

本人为前端程序员，这是第一次写后台程序，所以写的比较糙，单元测试和日志功能都没做。**可以学习练手使用，不可用于生产环境。**

### 技术栈

[egg](https://github.com/eggjs/egg/)
